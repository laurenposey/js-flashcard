JS and jQuery Flashcards

JavaScript

A language that adds functionality and interaction to websites.

Operators

Symbols that compare two items.

Variables

Words that can stand in for a function, string, or integers.

Variable naming conventions

Use a descriptive name.

Functions

Something that can return a result.

Methods

Built in functions.

Arguments

Functions can take arguments.

Parameters

Input needed to return a function.

Return

To return the product of the function to be used further.

Chaining methods

Multiple built-in functions that can chain to each other.

Strings

Characters with quotation marks that can be manipulated as words.

Booleans

True/False

Undefined

Variable without a value that can be used as a number.

NaN

An integer that is not a number.

Escape

To exit the command line.

Alert

A pop up.

Comments

Explanatory writing that doesn't affect the code.

jQuery

A JavaScript library.

Attributes

Something that modifies an HTML element (like href, src, width, etc.)
